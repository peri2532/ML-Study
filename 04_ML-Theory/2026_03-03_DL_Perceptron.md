# 1장 퍼셉트론 (Perceptron)
## 1. 퍼셉트론이란?
- 퍼셉트론은 신경망(딥러닝)의 기원이 되는 알고리즘이다.
- 다수의 입력 신호를 받아 하나의 출력 신호를 내보내는 구조를 가진다.
- 기본적으로 이진 분류(0 또는 1) 문제를 해결하는 모델이다.

## 2. 퍼셉트론의 구성 요소
- 입력 신호: $x_1, x_2$
- 출력 신호: $y$
- 가중치: $w_1, w_2$
- 임계값: $\theta$ (theta)
- 편향: $b$

- 신호의 의미
- 1 → 신호가 흐른다
- 0 → 신호가 흐르지 않는다

## 3. 퍼셉트론의 동작 원리
- 입력 신호가 뉴런(노드)에 전달될 때 각각 가중치가 곱해진다.
- 가중합이 임계값을 넘으면 1을 출력하고, 넘지 못하면 0을 출력한다.

- 수식 표현
- $w_1x_1 + w_2x_2$
- $y = 0 \quad (w_1x_1 + w_2x_2 \le \theta)$
- $y = 1 \quad (w_1x_1 + w_2x_2 > \theta)$
- 또는 편향을 포함하여 다음과 같이 표현할 수 있다.
- $y = w_1x_1 + w_2x_2 + b$

## 4. 가중치와 편향의 의미
- 가중치 (w)
- 각 입력 신호가 결과에 주는 영향력을 조절하는 매개변수값이 클수록 해당 입력이 결과에 더 큰 영향을 줌
- 전기 회로에서의 저항과 유사한 개념 (저항이 낮을수록 전류가 많이 흐르는 것처럼, 가중치가 클수록 신호 영향력이 큼)

- 편향 (b)
- 뉴런이 얼마나 쉽게 활성화될지를 조정하는 값
- 한쪽으로 치우쳐 균형을 깨는 역할
- 임계값 $\theta$를 변형한 형태

## 5. 퍼셉트론의 한계
- 퍼셉트론은 직선 하나로 데이터를 분리하는 선형 모델이다.
- 결정 경계는 항상 다음과 같은 형태이다.
- $w_1x_1 + w_2x_2 + b = 0$
- 따라서 선형 분리가 불가능한 문제는 해결할 수 없다.

## 6. XOR 게이트 문제
- XOR 진리표
- $x_1=0, x_2=0 \rightarrow 0$
- $x_1=0, x_2=1 \rightarrow 1$
- $x_1=1, x_2=0 \rightarrow 1$
- $x_1=1, x_2=1 \rightarrow 0$
- XOR은 직선 하나로 0과 1을 나눌 수 없다.
- 이를 선형 분리 불가능(linearly inseparable) 이라고 한다.
- 따라서 단층 퍼셉트론으로는 XOR을 표현할 수 없다.

## 7. 다층 퍼셉트론 (Multi-Layer Perceptron)
- 퍼셉트론을 여러 층으로 쌓은 구조
- 은닉층(hidden layer)을 추가하여 비선형 문제를 해결할 수 있다.

- 구조
- 입력층 → 은닉층 → 출력층
- 뉴런들이 서로 신호를 전달하며 최종 출력을 계산한다.
- 작업자(뉴런)들이 부품을 전달하는 과정과 유사하다.

## 8. XOR을 다층 퍼셉트론으로 표현하기
- XOR은 게이트 조합으로 만들 수 있다.
- $XOR = (A \text{ OR } B) \text{ AND } (A \text{ NAND } B)$
- AND, OR → 단층 퍼셉트론으로 구현 가능
- XOR → 2층 퍼셉트론 필요

- 동작 과정
- 1층: OR, NAND 계산
- 2층: AND 계산
- 최종 출력: XOR

## 9. 정리
- 퍼셉트론은 신경망의 기초가 되는 알고리즘이다.
- 여러 입력을 받아 하나의 출력을 생성한다.
- 단층 퍼셉트론은 선형 분리 문제만 해결 가능하다.
- XOR 문제는 단층 퍼셉트론으로 해결할 수 없다.
- 다층 퍼셉트론을 사용하면 비선형 문제를 해결할 수 있다.
