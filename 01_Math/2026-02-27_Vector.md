# 벡터의 내적과 외적
## 벡터의 내적

### 벡터의 내적 정의
-a·b = Σ a_i b_i  (성분끼리 곱해서 더한 값)

### 기하하적 의미
- 두 벡터가 얼마나 같은 방향을 바라보는가
- a⋅b=∥a∥∥b∥cosθ
    - 0 -> 직교(90도)
    - 양수 -> 비슷한 방향
    - 음수 -> 반대 방향

### 벡터의 내적 필요한 이유
- 각도 계산
- 직교성 판단
-  투영
- 길의(norm) 정의
- 거리 계산

## 벡터의 외적
### ※ 외적은 3차원에서만 정의 ※

### 벡터의 외적 정의
- 결과는 벡터
- a,b 모두에 수직
- 크기
    - ∥a×b∥=∥a∥∥b∥sinθ
    - 평행사변형의 넓이

### 반드시 이해해야할 포인트
    
    1. 내적은 "길이 + 각도"를 동시에 담는다
    - a⋅a=∥a∥^2
    - 여기서 norm이 정의되고 거리개념의 출발점이다.

    2. 직교는 ML에서 매우 중요하다.
    - a⋅b=0
    - PCA의 주성분은 서로 직교
    - SVD의 기저벡터는 직교
    - 최소제곱에서 오차벡터가 직교

    3. 외적은 "면적"을 만든다
    - 내적 -> 스칼라
    - 외적 -> 벡터

### ML / 통계 연결
-  코사인 유사도 cosθ=(a⋅b)​/(∥a∥∥b∥)
    - 텍스트마이닝에서:
        - 문서 벡터
        - TF-IDF 벡터
        - 임베딩 벡터

- MSE는 내적이다. 
$\|y - \hat{y}\|^2 = (y - \hat{y})^T (y - \hat{y})$
    - 벡터 길이의 제곱
    - MLE(정규분포 가정)과 직접 연결

- PCA에서의 내적 의미
    -공분산 행렬:X$^T$X
    - 각 원소는 데이터 벡터 사이의 내적
    - 즉, PCA는 내적 구조를 분석하는 방법

※ 내적은 공간의 거리와 각도를 정의하는 연산이며,외적은 3차원에서 면적과 수직 방향을 정의하는 연산이다.

## 추가 개념 (ML 연결)

### 투영(Projection)
- 정의: 벡터를 어떤 방향/부분공간에 가장 가깝게 붙이는 것
- 공식: proj_a(y) = (a·y)/(a·a) * a
- 핵심: y - proj_a(y)는 a와 직교

### 최소제곱(Least Squares)과 직교
- 목표: min ||y - Xβ||^2
- 최적조건: X^T (y - Xβ) = 0  (오차 e는 column space에 직교)
- 해석: y를 X의 열공간에 직교투영한 것이 Xβ

### MSE와 MLE 연결
- ||y - ŷ||^2 = (y-ŷ)^T(y-ŷ)
- 정규분포 오차 가정 시 MLE 최대화 ↔ 제곱오차 최소화

### 문서 벡터 / TF-IDF / 임베딩
- 문서벡터: 문서를 숫자 벡터로 표현
- TF-IDF: 단어 빈도 기반(희소·고차원)
- 임베딩: 의미 기반(밀집·저차원)

### 코사인 유사도
- cosθ = (a·b)/(||a|| ||b||)
- 크기 영향을 제거하고 방향 유사도만 비교
